 # product-title-classifier 
Ovaj projekat razvijam sa ciljem da napravim model mašinskog učenja koji automatski predlaže kategoriju proizvoda na osnovu njegovog naslova.

## Struktura projekta
- **notebooks/** — istraživanje podataka, analiza i treniranje modela  
- **src/** — Python skriptovi (treniranje i predikcija)  
- **data/** — skup podataka (products.csv)  
- **docs/** — dokumentacija i slike
